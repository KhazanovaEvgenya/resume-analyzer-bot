# Resume Analyzer Telegram Bot

Telegram-бот для анализа резюме под конкретную вакансию.

Пользователь отправляет PDF с резюме, затем ссылку на вакансию hh.ru или текст вакансии. Бот сравнивает резюме с вакансией и даёт рекомендации.

## Структура проекта

- `bot.py` — основной телеграм-бот на `aiogram`
- `config.py` — загрузка `TELEGRAM_BOT_TOKEN` и `PROXY_API_KEY`
- `llm.py` — вызов модели через ProxyAPI
- `pdf_utils.py` — извлечение текста из PDF
- `vacancy_utils.py` — получение текста вакансии из URL hh.ru или использование переданного текста
- `requirements.txt` — зависимости
- `.env.example` — пример настроек окружения

## Установка

1. Установите зависимости:

```bash
python -m pip install -r requirements.txt
```

2. Создайте файл `.env` на основе `.env.example` и заполните:

```text
TELEGRAM_BOT_TOKEN=ваш_токен_бота
PROXY_API_KEY=ваш_proxyapi_ключ
```

## Запуск

```bash
python bot.py
```

## Использование

1. Откройте чат с ботом в Telegram @resumeaiiiiiiibot
2. Отправьте `/start`
3. Отправьте PDF резюме
4. Отправьте ссылку на вакансию hh.ru или текст вакансии
5. Получите анализ и рекомендации

## Примечания

- Проект использует ProxyAPI для доступа к OpenAI API.
- Если PDF содержит только изображения, текст может не извлечься.
- Для hh.ru вакансий бот пытается парсить описание страницы автоматически.
