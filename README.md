# Resume Analyzer Telegram Bot

Простой Telegram-бот для анализа резюме в PDF. Пользователь отправляет резюме, бот извлекает текст и делает оценку с рекомендациями.

## Установка

1. Клонируй или создай проект в папке `resume-analyzer-bot`
2. Установи зависимости:

```bash
python -m pip install -r requirements.txt
```

3. Создай файл `.env` на основе `.env.example` и добавь токены:

```text
TELEGRAM_BOT_TOKEN=ваш_токен_бота
OPENAI_API_KEY=ваш_api_ключ_openai
OPENAI_MODEL=gpt-3.5-turbo
```

## Запуск

```bash
python bot.py
```

## Как пользоваться

1. Открой чат с ботом в Telegram
2. Отправь команду `/start`
3. Отправь файл резюме в PDF
4. Получи анализ и оценку

## Комментарии

- Бот использует OpenAI для анализа текста резюме.
- Если PDF содержит только изображения, извлечение текста может не сработать.
- Можно заменить модель на другую в `.env`.
